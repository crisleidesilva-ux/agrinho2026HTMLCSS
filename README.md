# Dashboard Rural - Controle de Estoque de Ração

## Sobre o projeto

O **Dashboard Rural - Controle de Estoque de Ração** é um sistema simples para registrar entradas, saídas e saldo de ração em uma propriedade rural.

Diferente de uma tabela comum, este projeto usa visual de painel administrativo, com indicadores e barras de resumo.

Tema:

**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente**

## Funcionalidades

- Registro de entrada de ração.
- Registro de saída de ração.
- Cálculo automático do saldo.
- Indicadores de entrada, saída e saldo.
- Gráfico simples com barras.
- Histórico das movimentações.
- Alerta de estoque baixo ou zerado.
- Layout responsivo.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Como usar

1. Baixe o projeto.
2. Abra a pasta no VS Code.
3. Abra o arquivo `index.html` no navegador.
4. Digite o tipo de ração.
5. Escolha entrada ou saída.
6. Informe a quantidade.
7. Clique em **Registrar movimentação**.

## Objetivo educativo

Mostrar como a tecnologia pode ajudar o produtor rural a organizar melhor seus recursos, evitar desperdícios e manter o controle da alimentação dos animais.
